# Blood Bank Management System

A C++ based Blood Bank Management System developed to manage donor records, blood stock, and patient requests efficiently. This project uses Object-Oriented Programming and file handling concepts to create a simple menu-driven management system.

## Features
- Add donor records
- Search blood groups
- Update and delete records
- Manage blood availability
- Patient request handling
- File handling support

## Technologies Used
- C++
- OOP Concepts
- File Handling
